# startup-leads

Local-first CLI for discovering job-search leads.

See [`docs/startup-leads-spec-final.md`](docs/startup-leads-spec-final.md) for the full specification.

## Quick start

```bash
bun install
bun test
bun run src/cli/index.ts --help
```
